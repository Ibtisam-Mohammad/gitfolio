# **App Name**: Gitfolio

## Core Features:

- GitHub Profile Retrieval: Fetch a user's GitHub profile data using the provided username.
- Repository Analysis: Analyze the user's GitHub repositories to extract key information such as primary languages, contribution frequency, and project stats.
- Portfolio Generation: Automatically generate a personalized portfolio website based on the GitHub data.
- Live Preview: Provide a live preview of the generated portfolio website.
- Theme Customization: Allow users to customize the theme of their portfolio via a configuration file.
- Resume integration: Allows the user to enhance their portfolio with their personal resume
- AI Professional Summary Generation: Leverage a tool to generate a professional summary for the portfolio, based on code patterns and contribution history.

## Style Guidelines:

- Primary color: Use a saturated orange (#F26419) inspired by the imagery's graphical components. 
- Background color: Use an off-white, very lightly desaturated background (#FAF9F6), as seen on the website screenshot.
- Accent color: Use a very dark grayish-yellow (#B5A642) to highlight certain UI elements.
- Headline font: 'Space Grotesk' (sans-serif) for titles, section headers, and emphasis;  Body font: 'Inter' (sans-serif) for readability.
- Employ a consistent style of simple outlined icons to represent technologies, skills, and project categories. Try to add unique twist to each one as depicted in the image. 
- Utilize a clean, grid-based layout with ample white space to present the information clearly. Arrange projects as cards or tiles to provide a visually appealing structure.
- Add subtle transitions and animations for loading new sections.
